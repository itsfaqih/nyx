# nyx
Nyx is a Free Portfolio Template made with TailwindCSS.
This template is currently in development.

## Features
- [x] Dark themed
- [x] Hero section
- [x] Works section with carousel
- [x] Testimonies section with carousel
- [x] Pricing card
- [x] Contact in footer
